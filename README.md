Features

Randomly selects a word from a predefined list

Provides hints for each word

Displays word progress with underscores

Visual ASCII hangman that updates with wrong guesses

Tracks guessed letters and prevents repeated guesses

Interactive win/lose messages

Requirements

Python 3.x

No additional libraries required


------------------->
How to Play

Run the game:

python hangman.py


Guess the word one letter at a time.

Check your progress and remaining attempts.

Use hints to help you guess.

The game ends when the word is guessed or the hangman is complete.
